# 作品集网页 — Spec

> 生成时间: 2026-06-10 | 工具: grill-me 脑暴

---

## 1. 产品定位

| 决策项 | 结论 |
|---|---|
| 受众 & 目标 | C — 初筛快扫 + 入围细看，兼顾保研/求职/实习复用 |
| 双层架构 | 表层抓眼球（30秒内留下印象），深层有内容（展开后每个项目独立叙事） |

## 2. 技术方案

| 决策项 | 结论 |
|---|---|
| 技术栈 | Next.js (App Router) + Tailwind CSS |
| 部署 | Vercel（免费，自带 Analytics） |
| 动画库 | Framer Motion（页面过渡）+ GSAP ScrollTrigger（滚动驱动）+ Three.js（背景粒子点缀） |
| 字体 | 精选中文字体 subset + Inter/SF Pro 英文 |
| 响应式 | 桌面优先，Tailwind 断点自然适配移动端 |

## 3. 视觉设计

| 决策项 | 结论 |
|---|---|
| 风格 | Apple 产品页——极致克制、大字号、大留白、滚动叙事 |
| 色彩 | 深色主题 (`#000` 纯黑背景，白字) |
| 语言 | 中文叙事 + 英文关键词标签 |
| 动效 | C — 沉浸式：页面过渡 + 卡片 hover 鼠标跟随 + 视差滚动 + Three.js 背景粒子点缀 + GSAP ScrollTrigger |

## 4. 信息架构

| 决策项 | 结论 |
|---|---|
| 页面结构 | 首页 + 项目详情页（分节叙事） |
| Hero | 大标题宣言式（名字 + 「作品集」+ 副标题），极致克制 |
| 导航 | 极简顶栏毛玻璃：「作品」「关于」+ GitHub 图标 |
| 首页精选 | 3 张核心卡片：《代码：升变》《疫城札记》ViSmoke |
| 更多项目 | 小网格卡片（银星劫/Dove Simulator/课程项目/Kaggle）|
| About | 技能标签云 + 获奖列表，碎碎念放底部角落，简历附下载 |
| Footer | 社交链接（GitHub/邮箱）+ 返回顶部 |

## 5. 项目详情页

| 决策项 | 结论 |
|---|---|
| 结构 | C — 分节叙事，每节独立一屏，滚动驱动动画连接 |
| 每节内容 | Hero 标题 / 核心亮点 / 技术细节 / 截图画廊 / 外部链接 |
| 外链 | 比赛官网 / GitHub / Demo，带 utm 追踪 |

## 6. 跳转追踪
