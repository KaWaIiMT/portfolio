## 以后如何更新

Clash 代理已配好（git 走 http://127.0.0.1:7897），以后每次改完内容只需要两步：

```powershell
# 1. 构建
cd "D:\_College\保研\作品集"
npx next build

# 2. 推送
git add -A
git commit -m "描述你的改动"
git push
```

GitHub 会在推送后自动从 docs/ 文件夹部署，1-2 分钟生效。

## 如果换 VPN 或代理端口变了

```powershell
# 查看当前代理
git config --global http.proxy

# 改成新端口
git config --global http.proxy "http://127.0.0.1:7890"
git config --global https.proxy "http://127.0.0.1:7890"

# 取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy
```
